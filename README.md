## Installation:

1. Clone repo
1. Set up environment variables
   ```
   cp .env.local.example .env.local
   ```
1. Enter your data into .env.local
1. run `npm intall`

## Usage:

1. run `yarn run dev` to start app
1. Navigate to `localhost:3000`
